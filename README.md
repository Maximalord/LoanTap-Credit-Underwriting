# LoanTap Personal Loan Credit Underwriting

This project focuses on building a credit underwriting model for personal loans using Logistic Regression.

## Key Highlights
- Exploratory Data Analysis (EDA)
- Feature Engineering & Encoding
- Class Imbalance Handling
- Logistic Regression with Threshold Tuning
- ROC-AUC & Precision-Recall Evaluation
- Business-driven Risk Segmentation

## Business Objective
Predict borrower creditworthiness and recommend risk-based loan approval strategies while minimizing NPAs.

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
